An easy to use CMS with In Content Editing skills

ICE includes a revolutionary new CMS that plugs directly into your site. Essentially, the CMS puts the editor right there in your design, right by the content. The user is editing right in your page, seeing changes in real time.

ICE is made to be easy to use, and fast to set up. Designers and developers can use their existing workflow to design the site using regular web technologies, because you only need three lines to transform a HTML-file into a template.

<?php
include 'ice/ice-adapter-php';
$ice->load('pagename'); ?>
...

<head>
<?php $ice->head(); ?>
</head>

<!-- To create a editable div, with the class "big" -->
<?php element("id", "tag", "type", array("class"=>"big")); ?>

How do I get it?

  1. Clone git://github.com/mattenrone/ICE.git
  2. Place the "ice" folder in the root of your project.
  3. Browse to the installer in root/ice/install/ and follow the instructions.
  4. Log in to the administration and add the templates using "add files" under the advanced menu.
  5. Done.

Be aware that ICE is in the pre-alpha stage.
